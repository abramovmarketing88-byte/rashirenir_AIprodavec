# Лендинг с отправкой заявок в Telegram

## Быстрый запуск

1. Создайте `.env` на основе `.env.example` и заполните:
   - `TG_TOKEN` — токен вашего бота.
   - `TG_CHAT_ID` — числовой ID вашего Telegram-чата.
2. Запустите сервер:

```bash
set -a
source .env
set +a
npm start
```

3. Откройте `http://localhost:3000`.

## Важно по безопасности

Не храните токен бота в `index.html` и не публикуйте его в открытом репозитории. Токен должен находиться только на сервере в переменных окружения.
